Route simple tasks to Haiku ($0.80/M), complex to Sonnet ($3/M), critical to Opus ($15/M)

2. Prompt Caching (90% off)

Cache system prompts - reduces input costs by 90% on repeated calls

3. Batch API (50% off)

Use for async workloads - evals, reports, bulk analysis

Context management - rolling summarization at 80% limit

Tool error handling - retry with exponential backoff
Model version pinned - not using 'latest'
Golden eval set - 50+ test cases verified
Output validation - schema validation with auto-retry
Safety evals - adversarial test suite passing
Observability - traces & logs to monitoring platform
Cost limits - per-request budget enforced

🏗️ Multi-Agent Topologies

Orchestrator → Sub-Agent (Most Common)

One orchestrator delegates to specialized sub-agents. Best for complex tasks with clear sub-tasks.

Generator → Critic Loop

Generator creates, critic evaluates, iterate. Excellent for code review, content generation.

Blackboard (Shared State)

Agents read/write to shared store. Best for async pipelines, long-running tasks.

🛡️ Safety & Security

Prompt Injection Defense

Sanitize tool outputs, use structural delimiters, implement least privilege, monitor for anomalies.

Human-in-the-Loop

Always require approval for: sending external emails, deleting data, financial transactions, public posts.
